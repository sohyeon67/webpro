a 태그 : 페이지이동  을 위한 태그
			이동할 페이지를 href 속성에 설정한다
			target : 이동페이지를 새로운 탭 또는 현재탭에서 실행한다
			기본값은 target = "_self" 현재탭에서 실행한다
			
			a:link => 처음 만들어질 때
			a:visited => 한번 방문했을 때
			a:hover => 마우스 올려놓을 때
			a:active => 누르는 순간
			
			밑줄을 나타내는 속성
			text-decoration : underline / overline / none / line-through
